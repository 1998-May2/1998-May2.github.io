芒果视频转换格式

最近下了一点芒果TV的视频，然后发现视频文件后缀是.MTV，想要改成通用格式就搜了一下，发现确实可以改，还可以用bat批处理指令来改，🐎了

ren *.mtv *.mp4

参考：https://mp.weixin.qq.com/s/JZQ8hr_pSQLBbrcTdMyS0Q

更改右键菜单

右键「开始菜单」（快捷键 Windows + X）找到「Windows 终端（管理员）」

reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve

重启电脑

恢复右键菜单

reg.exe delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /va /f

重启电脑
