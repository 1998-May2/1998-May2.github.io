使系统变得更好的设置

转自国光https://www.sqlsec.com/2022/01/monterey.html

设置启动台图标数量

[tip type="blue"]设置列数为 10[/tip]

defaults write com.apple.dock springboard-columns -int 10

[tip type="blue"]设置行数为 5[/tip]

defaults write com.apple.dock springboard-rows -int 5

[tip type="blue"]重启 Dock 生效[/tip]

killall Dock

恢复启动台的默认数量

[tip type="blue"]恢复默认的列数和行数[/tip]

defaults write com.apple.dock springboard-rows Default
defaults write com.apple.dock springboard-columns Default

[tip type="blue"]重启 Dock 生效[/tip]

killall Dock

减少程序坞的响应时间

[tip type="blue"]设置启动坞动画时间设置为 0.5 秒[/tip]

defaults write com.apple.dock autohide-time-modifier -float 0.5 && killall Dock

[tip type="blue"]设置启动坞响应时间最短[/tip]

defaults write com.apple.dock autohide-delay -int 0 && killall Dock

[tip type="blue"]恢复启动坞默认动画时间[/tip]

defaults delete com.apple.dock autohide-time-modifier && killall Dock

[tip type="blue"]恢复默认启动坞响应时间[/tip]

defaults delete com.apple.Dock autohide-delay && killall Dock
